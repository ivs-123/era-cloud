# ERA Cloud: Business Canvas

## 1. Что мы продаём

**ERA Cloud — универсальный облачный гейт.** Один API-ключ, единый баланс, автоматическая маршрутизация на лучший инференс/GPU по цене, скорости или надёжности.

```
Клиент → [ERA Cloud Gateway] → Автовыбор из 40 провайдеров → Самый дешёвый/быстрый инференс
```

Клиенту не нужно: регистрироваться у 40 провайдеров, мониторить цены, писать fallback-логику, бояться vendor lock-in.

## 4. Модель дохода

### Поток A: Маржинальная перепродажа (SMB/SME)
```
Клиент платит $1.00 за 1M токенов Llama-3.3-70B
ERA Cloud покупает у Lepton за $0.32
Маржа: $0.68 (68%)
```
**Прогноз:** 100 клиентов × $500/мес = $50,000 MRR, маржа ~55% → $27,500/мес чистыми.

### Поток B: Enterprise BYOK (Фиксированная SaaS-подписка)
```
Клиент вводит свои ключи AWS/GCP/Yandex
ERA Cloud оптимизирует маршрутизацию
Подписка: $500-2,000/мес
```
**Прогноз:** 20 enterprise × $1,000/мес = $20,000 MRR, маржа ~90%.

### Поток C: Реферальные отчисления
```
Клиент ERA Cloud → покупает GPU у ThunderCompute
ThunderCompute платит ERA Cloud 3% от трафика
```
**Прогноз:** $100K трафика × 3% = $3,000/мес пассивно.

### Поток D: Гранты (недоходный, инфраструктурный)
```
AWS Activate: $100,000 облачных кредитов
GCP for Startups: $350,000 облачных кредитов
RunPod Startup: $75,000 GPU-кредитов
Alibaba AI Catalyst: $120,000
```
**Итого:** до $645,000 бесплатной инфраструктуры на запуск.

## 5. Конкуренты и отличие

| Конкурент | Что делает | Наше отличие |
|-----------|-----------|-------------|
| **OpenRouter** | Роутинг LLM (США) | Мы: + GPU-серверы, + РФ/Азия, + Edge |
| **LiteLLM** | Open-source прокси | Мы: managed SaaS + 40 провайдеров |
| **Portkey** | API-гейт | Мы: + инфраструктурный слой (GPU) |
| **Together AI** | Inference API | Мы: агрегатор, а не провайдер |
| **nebius.ai** | GPU-облако | Мы: мультипровайдерный гейт |

**Наше уникальное преимущество (USP):** Гибридный гейт — публичные API + приватные GPU-серверы + РФ-провайдеры + азиатский рынок. Никто не закрывает все 4 слоя одновременно.

1. **Получить API-ключи:** DeepInfra, Groq, ThunderCompute, Cloudflare
2. **Прогнать живой e2e:** Thunder → создать GPU-инстанс → инференс → стоп → инвойс

5. **Гранты:** подать заявки AWS Activate + GCP Startups от грузинского юрлица
6. **Деплой:** `eracloud.pro` с API + дашбордом
